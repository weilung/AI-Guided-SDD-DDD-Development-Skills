# Proposals

This directory is for active proposal work only.

- Use `PROPOSAL-000-template.md` when drafting a new proposal.
- Keep draft / evaluating / approved proposals here while they are active.
- Move implemented, rejected, or superseded proposals to `archive/proposals/` after closeout.

Historical proposal records from P001-P015 were archived on 2026-05-01 after the first public npm publish.
